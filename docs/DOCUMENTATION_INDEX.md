## 🎓 关键概念速查

### 公式引擎
- **概念**: hot-formula-parser 库的封装
- **限制**: 变量名不能含数字 (B2 不行，x 可以)
- **解决**: 值替换策略 + 递归计算
- 📖 详见: [FIX_RECURSIVE_FORMULA.md](./FIX_RECURSIVE_FORMULA.md)

### 缓存机制
- **概念**: 存储已计算的公式结果
- **关键**: 循环引用检测 + 缓存失效
- **好处**: 性能提升 10 倍以上
- 📖 详见: [CACHE_OPTIMIZATION.md](./CACHE_OPTIMIZATION.md)

### 虚线框
- **概念**: Canvas 绘制的虚线矩形
- **样式**: 5px 线 + 5px 间距，绿色 (#10b981)
- **用途**: 选择范围视觉反馈
- 📖 详见: [PHASE3_DASHED_BOX_TEST.md](./PHASE3_DASHED_BOX_TEST.md)

### 快捷键
- **Shift+方向键**: 扩展选择范围
- **Ctrl+A**: 全选表格
- **Escape**: 清除选择
- 📖 详见: [PHASE3_KEYBOARD_SHORTCUTS.md](./PHASE3_KEYBOARD_SHORTCUTS.md)

### 性能优化
- **方案**: requestAnimationFrame 限制重绘
- **效果**: 60fps 稳定，CPU 下降 60%
- **关键**: 避免频繁同步重绘
- 📖 详见: [PHASE3_PERFORMANCE_OPTIMIZATION.md](./PHASE3_PERFORMANCE_OPTIMIZATION.md)
