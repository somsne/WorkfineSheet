## 📝 文档规范

### 命名规则
- **大写蛇形**: `FEATURE_NAME.md` - 主要文档
- **小写中划线**: `feature-name.md` - 进度文档
- **PascalCase**: `ComponentName.md` - 组件文档

### 文档模板
每个功能文档应包含：
- 功能描述
- 实现方案
- API 说明
- 使用示例
- 测试验证
